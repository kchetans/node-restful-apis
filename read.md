
## Development setup

Required Nodejs 8.X, NPM 5.X

set node env to development

```
$ chmod +x  ./setup.sh
$ ./setup.sh 
```



## Development Build

```
$ npm start
```

## Production Build

```
$ npm run build
$ npm run serve
```
